☕java speciality enti ante anni Languages la idi direct ga object code ni produce cheyyadu:

  1. JDK, program source code ni compile chestundi, Byte code(low level code) ni produce chestundi.
  2. Produce ayina byte code ki pc software ki yelanti relationship undadhu.
  3. Produce ayina byte code ni JRE lo JVM execute chestundi.
  4. Byte code ye operating system lo run ayina same output vastundi.
  5. Write Once Run Anywhere, softwares different ayina sare na pc lo rasina code naku and nee system lo kooda same output vastundi.
  6. So that's why java is Platform Independent Programming Language,
