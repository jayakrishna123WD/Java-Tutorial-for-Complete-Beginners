☕ indexOf() Method --- Returns the position of the first found occurrence of specified characters in a string
☕ ee method lo 4 types unnay, those methods are
★ indexOf(String str) -- Returns mentioned string first occurrence in string
★ indexOf(String str, int fromIndex) -- Returns mentioned string which occurs from mentioned index
★ indexOf(int char) -- Returns mentioned character first occurrence in string
★ indexOf(int char, int fromIndex) Returns mentioned character which occurs from mentioned index

JAVA


System.out.println(name.indexOf("Name"));


// 5


System.out.println(name.indexOf("ur", 4));


// -1


//index 4 taruvata ur occurrence ledu kadha anduke -1 vacchindi


System.out.println(name.indexOf('r'));


// 3


System.out.println(name.indexOf('m', 5));


// 7


//index 5 nunchi m occurrence index value 7 deggara ayyindi kadha anduke output 7

JAVA


public class MyClass {


public static void main(String[] args) {


System.out.printf("My name is %s \n", "ne name" );


System.out.printf("My age is %d \n", ne age);

System.out.printf("My height is %.1f \n", ne height);


}


}

output

My name is ne name
My age is ne age
My gender is ne gender
My height is ne height
