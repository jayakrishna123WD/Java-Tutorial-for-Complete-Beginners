☕ Polymorphism, having many forms. one method many impementations. There are two types of polymorphism

⭐ compile-time polymorphism

★ Another name is Method Overloading, Methods have same name having different parameters

⭐ runtime polymorphism

★ Another name is Method Overriding
★ Methods must maintain same return type and parameters, access level should be public

☕ movie lo hero ki multiple personality disorder untundi ga, ante oka person ki three different personalities untay alane polymorphism kooda okate method name but different implementation.

⭐ Method Overriding

JAVA


class Ramu {


public void Person() {


System.out.println("Nenu ramu");


}


}


class Remo {


public void Person() {


System.out.println("I am REMO");


}


}


class Aparichitudu {


public void Person() {


System.out.println("Nenu Aparichitudu");


}


}


public class MyClass {


public static void main(String[] args) {


Ramu ramu_personality1 = new Ramu();


ramu_personality1.Person();

}


}

output

Nenu ramu

public class MyClass {


public static void main(String[] args) {


Remo ramu_personality2 = new Remo();


ramu_personality2.Person();

}


}

output

I am REMO

public class MyClass {


public static void main(String[] args) {


Aparichitudu ramu_personality3 = new Aparichitudu();


ramu_personality3.Person();


}


}

output

nenu aparichitudu

☕ method name okate kani mana requirement batti yenni sarlu ayina diferent purposes ki implement cheskovacchu
☕ above code is example for method Overriding because Person method access level is public and anni methods ki same return type undi.

⭐ Method Overloading

JAVA


public class MyClass {


static int calculate(int one) {


return 5 + one;


}


static int calculate(int one, int two) {


return one + two;


}


static int calculate(int one, int two, int three) {


return one + two + three;


}


public static void main(String[] args) {


int sum = calculate(5);


int sum2 = calculate(5, 4);


int sum3 = calculate(5, 4, 10);


System.out.print(sum + " " + sum2 + " " + sum3);


}


}

output

10 9 19

☕ The above code relates to method Overloading, method name okate but different parameters
