☕ Java lo chala use ayye konni math methods unnay avento choodam randi
☕ Math kooda oka class ye, but ah class lo undevi static methods kabatti no need to create object, direct ga math class lo unna methods ni access cheskovacchu
☕ appudu java syntax lo migilinadi taruvata cheptha anna ga, ah time vacchindi. System ani choosaru kadha adhi kooda oka class ye andulonu static methods unnay kabatte object create cheyyaledu direct ga system lo unna methods ni use chestunnam. static keyword kosam time vacchinappudu cheppukundam

☕ Absolute value

★ value negetive ayina sare output positive la vastundi

JAVA


byte value = -19;


System.out.println("Number is: "+ Math.abs(value));


byte value = 19;


System.out.println("Number is: "+ Math.abs(value));


output

Number is: 19

Number is: 19

☕ Floor

★ return the number which is down to the nearest whole number

JAVA


float value = 13.46f;


System.out.println("Number is: "+ Math.floor(value));


output

Number is: 13.0

☕ ceil

★ return the number which is up to the nearest whole number

JAVA


float value = 13.46f;


System.out.println("Number is: "+ Math.ceil(value));


output

Number is: 14.0

☕ Natural Logarithm

★ This method returns Logarithm of a number

JAVA

☕ Maximum

★ This method returns the maximum of two numbers

JAVA


byte value1 = 43;


byte value2 = 83;


System.out.println(Math.max(value1, value2));


output

83

☕ Minimum

★ This method returns the minimum of two numbers

JAVA


byte value1 = 89;


byte value2 = 34;


System.out.println(Math.min(value1, value2));


output

34

☕ Power

★ This method return a value which is risen to the power of another number. simply exponential operation ni perform chestundi

JAVA


byte value1 = 2;


byte value2 = 3;


System.out.println(Math.pow(value1, value2));


output

8.0

☕ Random

★ This method returns random number in double datatype we need to convert it to int

JAVA


double value = Math.random()*10;


System.out.println(Math.random());


System.out.println((int)value);


output

0.3672904679874903
2

★ double datatype value ni int la convert chesi print chesam. we'll discuss about type conversion in coming chapters.

☕ Square Root

★ This method will returns square root of a number

JAVA


int value = 144;


double result = Math.sqrt(value);


System.out.println(result);


System.out.println((int)result);


output

12.0
