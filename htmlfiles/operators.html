☕ Arithmetic Operators
  1. (+) --- Addition
  2. (-) --- Subtraction
  3. (*) --- Multiplication
  4. (/) --- Division
  5. (%) --- Modulo Operation (Remainder after division)

JAVA


public class MyClass {


public static void main(String[] args) {


byte total = 8 + 2; //total = 10


total = 8 - 2; //total = 6


total = 8 * 2; //total = 16


total = 8 / 2; //total = 4


total = 8 % 2; //total = 0 (Remainder zero)


}


}

☕ Relational Operators
  1. == --- Is Equal To
  2. != --- Not Equal To
  3. > --- Greater Than
  4. < --- Less Than
  5. >= --- Greater Than or Equal To
  6. <= --- Less Than or Equal To

JAVA


public class MyClass {


public static void main(String[] args) {


byte number_one = 10, number_two = 20;


System.out.println(number_one == number_two);


System.out.println(number_one != number_two);


System.out.println(number_one > number_two);


System.out.println(number_one < number_two);


System.out.println(number_one >= number_two);


System.out.println(number_one <= number_two);


}


}

☕ Logical Operators
  1. && --- Logical AND (operands true aythene output true vastundi)
  2. || --- Logical OR (oka operand true ayina output true vastundi)
  3. ! --- Logical NOT (operand value reverse avtundi, true--false avtundi and false--true avtundi

JAVA


public class MyClass {


public static void main(String[] args) {


// && Operator


System.out.println((10 > 7) && (20 == 20));


System.out.println((35 < 15) && (20 != 30));


// || Operator


System.out.println((20 > 45) || (20 > 7));


System.out.println((20 < 45) || (20 > 7));


System.out.println((30 == 45) || (20 > 56));


// ! Operator


System.out.println(!(30 == 45));


System.out.println(!(30 < 45));


}


}

output

true //true and true results true
false // false and true results false
true // false or true results true
true // true or true results true
false // false or false results false
true //not false results true
false // not true results false

☕ Unary Operators
  1. + --- Unary plus
  2. - --- Unary minus
  3. ++ --- Increment operator
  4. -- --- Decrement operator
  5. ! --- Logical complement operator

JAVA


public class MyClass {


public static void main(String[] args) {


byte total = 10; //total = 10


// postfix (variable++, variable--)


System.out.println(total++); //output-10, total = 11


System.out.println(total--); //output-11, total = 10


// prefix (++variable, --variable)


System.out.println(++total); //output-11, total = 11


System.out.println(--total); //output-10, total = 10


}


}

☕ Ternary Operator
  1. ? :
  2. variable = condition ? expression1 : expression2
if condition is true expression1 is stored in variable
if condition is false expression2 is stored in variable.
☕ Java Operators precedence

Ternary Operator example:

JAVA


public class MyClass {


public static void main(String[] args) {


boolean choice = 5 > 3;


System.out.println(choice ? "5" : "3");


}


}

5
