Break:

☕ last code lo even numbers ni print chessam kadha aythe naku anni vaddu nak antha memory power ledu yedo chinnapiladni 2 aythe gurtuntundi. So, two even numbers print chesaka for loop ni rest teesko mani cheppedam

JAVA


public class MyClass {


public static void main(String[] args) {


byte limit;


byte count;


for (limit = 10; limit > 0; limit--) {


if (count == 2) {


break;


}


else {


if (limit % 2 == 0) {


count++;


System.out.println(limit);


}


}


}


}


}

output

10
8

☕ traffic poilce yeppudeppu stop andama ani wait chesinattu paina unna code lo break statement kooda count variable yeppudu 2 avtundi ani wait chestundi. Final ga count 2 ayyaka break chestundi loop ni ante loop bayataki vacchestundi.

Continue:

☕ last code lo loop ni break cheyyadam nerchukunnam ippudu continue yela work avtundo choodam

JAVA


public class MyClass {


public static void main(String[] args) {


byte limit;


for (limit = 1; limit < 10; ++limit) {


if (limit % 2 == 0) {


continue;

}


else {


System.out.println(limit);


}


}


}


}

output

1
3
5
7
9
