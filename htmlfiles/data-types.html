JAVA Data Types:

☕ byte --- range from -128 to 127. Default value is 0
size --- 8-bit
☕ short --- range from -32,768 to 32,767. Default value is 0
size --- 16-bit
☕ int --- range from -2,147,483,648 to 2,147,483,647. Default value is 0
size --- 32-bit
☕ long --- range from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807. Default value is 0
size --- 64-bit
☕ float --- Default value is 0.0f
size --- 32-bit
☕ double --- Default value is 0.0d
size --- 64-bit
☕ boolean --- values are true & false. Default value is false
☕ char --- values, which are mentioned in between single quotes are char data type. Default value is '\u0000'
