Conditional & Switch Statements

⏪ Previous topic 🏠 Home ⏩ Next topic

☕ Oka particular condition satisfy ayinappudu required operation ni perform chesedhe ee Conditional & Switch Statements.

☕ if--else Syntax:

if (condition) {
operation1;
}
else {
operation2;
}

☕ if--else-if Syntax:

if (condition1) {
operation1;
}
else if (condition2) {
operation2;
}
else {
operation3;
}

☕ nested if--else-if Syntax:

if (condition1) {
if (condition2) {
operation1;
}
else {
operation2;
}
else if (condition3) {
operation3;
}
else {
operation4;
}

☕ Switch Syntax:

switch (expression) {
case 1:
//code;
break;
case 2:
//code;
break;
default:
//code;
break;
}

☕ condition true aythe if block work avtundi, condition false aythe else block work avtundi. condition yeppudu kooda if and else-if ke untundi, else ki condition undadu -- condition false aythe direct ga else block ye run avtundi.

JAVA


public class MyClass {


public static void main(String[] args) {


byte choclates = 5;


if (choclates > 4) {


if (choclates == 4) {


System.out.println("take 2 choclates");


}


else if (choclates == 3 || choclates == 2) {


System.out.println("only 1 choclate");


}


else if (choclates == 1) {


System.out.println("Do not take choclate");


}


else {


System.out.println("take 3 choclates");


}


}


else {


System.out.println("take all choclates");


}


}


}

output

take 3 choclates

☕ meeru kooda oka example teeskoni code la convert cheyyandi. try to understand the above code.
☕ else if anni sarlu rase badhulu simple ga Switch Statements tho manam cheyyochu.

JAVA


public class MyClass {


public static void main(String[] args) {


byte choclates = 3;


if (choclates > 4) {


switch (choclates) {


case 4:


System.out.println("take 2 choclates");


break;


case 3:


System.out.println("only 1 choclate");


break;


case 2:


System.out.println("only 1 choclate");


break;


case 1:


System.out.println("Do not take choclate");


break;


default :


System.out.println("take 3 choclates");


break;


}


}


else {


System.out.println("take all choclates");


}


}


}

output

take all choclates
