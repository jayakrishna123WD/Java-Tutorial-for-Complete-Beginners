☕ Enums -- enumerations, array like structure lo constants ni store cheyyadam. Collections of unchangable values.
☕ months in year, days in week unchangable kadha. if manaki constants tho requirement unte appudu enum ni use chestam.

JAVA


public class MyClass {


enum FrontEnd {


HTML,


CSS,


JAVASCRIPT


}


public static void main(String[] args) {


FrontEnd lang = FrontEnd.HTML


System.out.println(lang);

}


}

output

HTML

☕ printing all constant using Enhanced for loop(for each loop)

JAVA


public class MyClass {


enum FrontEnd {


HTML,


CSS,


JAVASCRIPT


}


public static void main(String[] args) {


for (FrontEnd lang : FrontEnd.values()) {


System.out.println(lang);


}


}

output

HTML
CSS
JAVASCRIPT

☕ meeru work chese code lo constant requirement unte alanti situations lo enums ni use cheskondi anni constants oka deggara untay kadha accessibility easy avtundi
