⭐ Single Inheritance

JAVA


class Dad {


String father_name = "your father name";


}


class You extends Dad {


String your_name = "your name";


}


public class MyClass {


public static void main(String[] args) {


You details = new You();


System.out.println(details.father_name);


System.out.println(details.your_name);


}


}

output

your father name
your name

☕ me object use chesi me class and me parent class lo details ni access chesam

⭐ Multilevel Inheritance

JAVA


class GrandFather {


String grandfather_name = "your grand father name";


}


class Dad extends GrandFather{


String father_name = "your father name";


}


class You extends Dad {


String your_name = "your name";


}


public class MyClass {


public static void main(String[] args) {


You details = new You();


System.out.println(details.grandfather_name);


System.out.println(details.father_name);


System.out.println(details.your_name);


}


}

output

your grand father name
your father name
your name

☕ Multilevel ante aradamaindi ga me grandparent info kooda me nunchi access cheyyochu

⭐ Hierarchical Inheritance

☕ meku ee doubt raleda, anni methods ni access antunnav kadha mari constructor method sangatenti. we can't inherit constructor method but we can invoke it from child classes using super keyword
☕ observe carefully

JAVA


class Details {


String full_name;


int age;


Details (String name, int age) {


this.full_name = name;


this.age = age;


}


void info () {


System.out.println("name: "+full_name+"\nage: "+age);


}


}


class Nuvvu extends Details{


Nuvvu (String name, int age) {


super(name, age);


}


}


class Nenu extends Details {


Nenu (String name, int age) {


super(name, age);


}


}


public class MyClass {


public static void main(String[] args) {


Nuvvu filldetails = new Nuvvu("Jayakrishna", 21);


Nenu filldetails2 = new Nenu("your name", your age);


filldetails.info()


filldetails2.info()


}


}

output

name: Jayakrishna
age: 21
name: your name
age: your age
