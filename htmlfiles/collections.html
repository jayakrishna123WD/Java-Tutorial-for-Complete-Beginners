☕ Java arrays tho vacchina problem yentante array size confirm ga declare cheyyali and extra elements add cheyyadam avvadu, unna elements ni delete cheyyadam avvadu, direct ga print cheyyadam avvadu. we can perform those features by using collection framework
☕ collections framework, which is in util package is collection of classes and interfaces which helps performing array operations
☕ ArrayList, LinkedList, Queue, HashSet, HashMap, etc,. are included in collections framework
☕ we can perform adding, removing getting of elements and size of element. checking an element present in array or not in an array with the help of collections framework
☕ lets see all useful data structures implementation and perform some operations

⭐ ArrayList

★ ArrayList use cheyyadam valla arrays size automatic ga adjust avtay. This is very similar to normal array and best for storing and accessing data.

JAVA


import java.util.ArrayList;


public class MyClass {


public static void main(String[] args) {


ArrayList<String> directions = new ArrayList<String>();


}


}

★ let's add directions into directions array

JAVA


directions.add("North");


directions.add("East");


directions.add("South");


directions.add("West");


System.out.println(directions);

output

[North, East, South, West]

★ accessing elements by index value using get method, don't forget that index value starts from 0

JAVA


System.out.println(directions.get(2));


//output is South

★ manipulate the data in directions array by index value and value by using set method

JAVA


directions.set(2, "North");


//arrays becomes [North(index 0), East(index 1), North(index 2), West(index 3)]

System.out.println(directions.get(2));


//output is North

★ remove element from directions array using remove method

JAVA


directions.remove(2);


System.out.println(directions);


output

[North, East, West]

★ size method will returns the size of the directions array

JAVA


System.out.println(directions.size());


// output is 3

★ find an element present in directions array or not by using contains method which returns boolean output

JAVA


System.out.println(directions.contains("South"));


// output is false


System.out.println(directions.contains("North"));


// output is true


★ printing the directions array elements using for each loop

for (String word : directions) {


System.out.print(word+" ");


}


output

North East West

⭐ LinkedList

JAVA


import java.util.LinkedList;


public class MyClass {


public static void main(String[] args) {


LinkedList<String> directions = new LinkedList<String>();


}


}

★ LinkedList is best for manipulating data in array as numerous inserts and deletes
★ if your requirement is upto for simple process of accessing and storing elements in array then choose ArrayList
★ if your requirement is for manipulating data for n times then choose LinkedList
★ Methods in ArrayList and LinkedList are common

⭐ HashSet

★ idhi kooda same as ArrayList and LinkedList, but vatiki order of data untundi deeniki oka sequence undadu random ga array elements ni print chestundi
★ HashSet is unordered collection, it doesn't have indexes. random ga values kavalante your choice will be HashSet and too it doesn't contain duplicate values
★ HashSet supports the methods add, remove, size, contains.

JAVA


import java.util.HashSet;


public class MyClass {


public static void main(String[] args) {


HashSet<String> names = new HashSet<String>();


}


}

JAVA


names.add("rakesh");


names.add("suresh");


names.add("rajesh");


System.out.println(names);

output

[suresh, rakesh, rajesh]

⭐ HashMap

★ HashMap kooda HashSet laane unordered collection of elements. HashMap lo data key value pair la store avtundi, HashMap array lo elements ni add or access cheyyalanna key use chesi access cheyyochu. key must be unique value in HashMap.

JAVA


import java.util.HashMap;


public class MyClass {


public static void main(String[] args) {


HashMap<String, Integer> amount = new HashMap<String, Integer>();


}


}

★ add elements to HashMap array using put method

JAVA


amount.put("hundred rupees", 100);


amount.put("fifty rupees", 50);


amount.put("ten rupees", 10);


System.out.println(amount);

output

{fifty rupees=50, ten rupees=10, hundred rupees=100}

★ accessing elements by key using get method

JAVA


System.out.println(amount.get("ten rupees"));


//output is 10

★ replace values using replace method

JAVA


amount.replace("ten rupees", 150));


System.out.println(amount.get("ten rupees"));


//output is 150

★ As like ArrayList HashMap also supports remove, size methods
★ find an element present in amount array or not by using contains method which returns boolean output

JAVA


System.out.println(amount.containsKey("hundred rupees"));


// output is true


System.out.println(amount.containsKey("North"));


// output is false


★ printing keys and values using for loop

JAVA


import java.util.Map;


import java.util.HashMap;


public class MyClass {


public static void main(String[] args) {


HashMap<String, Integer> amount = new HashMap<String, Integer>();


}


}

JAVA


amount.put("hundred rupees", 100);


amount.put("fifty rupees", 50);


amount.put("ten rupees", 10);


for (Map.Entry<String, Integer> amt : amount.entrySet() {


System.out.println(amt.getKey()+ " : " + amt.getValue());


}

output

fifty rupees : 50
ten rupees : 10
hundred rupees : 100

★ printing keys and values seperatly using for each loop

JAVA


for (String amt : amount.keySet() {


System.out.println(amt);


}

output

ten rupees
fifty rupees
hundred rupees

JAVA


for (int amt : amount.values() {


System.out.println(amt);


}

output

100
10
50

⭐ iterator

★ An Iterator is an object that can be used to loop through collections, like ArrayList and HashSet. It is called an "iterator" because "iterating" is the technical term for looping. To use an Iterator, you must import it from the java. util package.

JAVA


import java.util.HashSet;


import java.util.Iterator;


public class MyClass {


public static void main(String[] args) {


HashSet<String> names = new HashSet<String>();


}


}

JAVA


names.add("rakesh");


names.add("suresh");


names.add("rajesh");


Iterator<String> check = names.iterator();


while (check.hasNext()) {


System.out.println(check.next());


}

output

rajesh
rakesh
suresh
