private int balance = 10000;

}

return this.balance;

}

☕ balance variable lo manam munde cash enter chesesam kadha, ippudu using constructor method we can provide. basically ee constructor method, It can be used to set initial values for object attributes. same as others methods but it doesn't have return type.
