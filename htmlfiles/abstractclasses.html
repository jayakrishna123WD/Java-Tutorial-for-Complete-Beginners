☕ Abstract classes are created by using abstract keyword before class
☕ any class having abstract method then the class is abstract class
☕ we can't create objects directly for an abstract class, but we can create by inheriting abstract class using normal class then creating object for the normal class and we can access methods in abstract class

JAVA


abstract class Ram {


abstract void apps();


}


class SubwaySurf extends Ram {


void apps() {


System.out.println("I am installing");


}


}


public class MyClass {


public static void main(String[] args) {


SubwaySurf mobile = new SubwaySurf();


mobile.apps();


}


}

output

I am installing

☕ Important points to remember while working with abstract class

★ we can't create objects directly for an abstract classes
★ any class having abstract method is abstract class
★ by using another class we can inherit abstract class, creating object and using abstract class methods
