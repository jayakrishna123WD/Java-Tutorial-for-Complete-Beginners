☕ we can change the values in array using index value and we can sort the array and we can print array directly by using a method from Array class
☕ Array class lo static methods untay no need to create object, direct ga access cheskovacchu, but we need to import it before using.

JAVA


import java.util.Arrays;


byte[] numbers = {1, 2, 3};


numbers[2] = 5;


numbers[0] = 9;//numbers = {9, 2, 5}


System.out.println(Arrays.toString(numbers));


Arrays.sort(numbers)


System.out.println(Arrays.toString(numbers));


output

[9, 2, 5]
[2, 5, 9]

☕ if we want to import all classes methods in util package we've to write like
☕ import java.util.*; --- * means all classes

★ Multidimensional Arrays

☕ oka array lo multiple arrays unte adhe Multidimensional array, normal array ki single [] pettaru 2d array ki [][] pedtaru anthe.

JAVA


byte[][] elements = {{5, 9, 6}, {3, 1, 8}};


System.out.println(Arrays.toString(elements[0]));


System.out.println(Arrays.toString(elements[1]));


byte a = elements[1][0];


System.out.println(a);


output

[5, 9, 6]
[3, 1, 8]
3

☕ elements ane 2d array lo two arrays unnay oka array index 0 and second array index 1, individual arrays ki kooda index untundi kadha. ippudu 2d array lo first element kavalante second array index and andulo first element index kadha ante [index 1][index 0] ala access cheskuntam.
☕ same arrays lane indulo kooda index number use chesi data change cheyyochu.

⭐ For Each Loop

☕ for each loop is enhanced loop, which is useful in arrays lets print an array using this for each loop

JAVA


byte[] numbers = {1, 2, 3};


for (byte num : numbers) {


System.out.print(num+" ");


}


output

1 2 3

☕ simple ga untundi, easy ga use cheskovacchu. byte num endukante numbers array data type byte kadha anduke colon taruvata array petta so every loop ki num variable lo sequence lo array values untay vatini print statement tho print chestunnam anthe. so simple kadha meeru IDE open chesi try cheyyandi
