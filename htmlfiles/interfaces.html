☕ Interface ante complete ga abastract class lo only abstract methods ye untay.
☕ we can define interface class by using interface keyword before class
☕ Interfaces can't have constructor, and we cannot create objects to interface classes directly
☕ interface lo unde methods ki abstract keyword mention cheyyakarledu because interface antene all methods are abstract so, no need to mention abstrat before methods.
☕ one interface can extends another interface just like one class extends another class. In class we use extends keyword but in interface we use implements keyword
☕ Inheritance lo Multiple Inheritance support ledu annam kadha, we can perform that using interfaces. one class can extends two different interfaces at a time.
☕ All methods in interface have access level of public.

JAVA


interface Insta {


public void id();


}


interface Whatsapp {


public void number();


}


class You implements Insta, Whatsapp {


public void id() {


System.out.println("Your Id");


}


public void number() {


System.out.println("Your number");


}


}


public class MyClass {


public static void main(String[] args) {


You details = new You();


details.id();


details.number();


}


}

output

Your Id
Your number
