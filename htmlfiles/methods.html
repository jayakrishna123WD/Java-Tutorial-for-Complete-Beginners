☕ Functions ante meku telusu DRY(Do Not Repeat Yourself) ni follow avvadaniki use chestam
☕ aythe Procedural programming lo functions ni direct ga call chestam avi provide chesina data paina operation ni perform chestundi kadha
☕ but, Java total object oriented language antha classes kabatti manam object create chesi taruvata use cheskovacchu
☕ Methods lo total 4 ways lo use cheskovacchu

⭐ No return type No parameters
⭐ No return type but have parameters
⭐ Have return type No parameters
⭐ Have return type and also have parameters

☕ nenu static keyword kosam cheptha anna kadha, ah time vacchindi lets go 🤩

JAVA


public class MyClass {


void pilavandiSir() {


System.out.println("Namaskaram niranjan gaaru");


}


public static void main(String[] args) {


MyClass haha = new MyClass();


haha.pilavandiSir();


}


}

output

Namaskaram niranjan gaaru

☕ void means method em return cheyyadu. if two numbers addition ni return cheyyalante void place lo numeric data type pettukovali according to your size. if value is < 127 you can replace void by byte return type
☕ nen cheppa ga konni classes ki like Scanner, Array, Math, etc,. vatiki static methods untay no need to create objects. ippudu manam paina unna example ki static keyword tho cheddam

JAVA


public class MyClass {


static void pilavandiSir() {


System.out.println("Namaskaram niranjan gaaru");


}


public static void main(String[] args) {


pilavandiSir();


}


}

output

Namaskaram niranjan gaaru

☕ choosaru ga simple ga object cheyyakunda manam method ni access chesam. outside of main method lo unna static varibles ni kooda manam access cheyyocchu. static ante like google map la intlo undi world antha access cheyocchu anatlu static members ni access cheyyochu easy ga
⭐ ippude manam No return type No parameters way choosesam, lets go for second way No return type but have parameters

JAVA


public class MyClass {


static void calculate(int one) {


System.out.println(one + 5);


}


public static void main(String[] args) {


calculate(5);


}


}

output

10

☕ main method lo calculate(5) ani undi kadha akkada 5 ante argument, calculate method lo int one ani undi kadha adhi parameter
☕ manam method ni call chesinappudu 5 ane argument calculate method ki pass avtundi, parameter ah pass ayna argument ni teeskoni operation ni perform chestundi
⭐ let's go for Have return type No parameters way

JAVA


public class MyClass {


static byte calculate() {


return 5 + 6;


}


public static void main(String[] args) {


System.out.println(calculate());


}


}

output

11

⭐ let's go for Have return type and also have parameters way

JAVA


public class MyClass {


static int calculate(int one) {


return 5 + one;


}


public static void main(String[] args) {


System.out.println(calculate(8));


}


}

output

13
